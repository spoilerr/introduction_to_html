# Introduction to HTML

A beginner-friendly HTML project demonstrating basic website structure and navigation.

## Project Overview

This repository contains a simple two-page website built with HTML. It serves as an introduction to fundamental HTML concepts including document structure, semantic elements, navigation, and page linking.

## Files

- **index.html** - Main landing page featuring heading hierarchy, paragraph content, and navigation to the contact page
- **contact.html** - Contact information page with a link back to the home page

## Features

- Basic HTML5 document structure with proper semantic elements
- Multi-level heading hierarchy (h1-h4)
- Navigation between pages using button links
- Responsive meta viewport tag
- Header, main, and footer structural elements

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Use the navigation buttons to explore both pages

## Concepts Demonstrated

- HTML document structure (`<!DOCTYPE>`, `<html>`, `<head>`, `<body>`)
- Semantic elements (`<header>`, `<main>`, `<footer>`)
- Heading elements (`<h1>` through `<h4>`)
- Paragraph text (`<p>`)
- Links and navigation (`<a>`, `<button>`)
- Meta tags for character encoding and viewport settings
